# Prompts

> Versioned AI prompts used in this project. Treat these like code — iterate, test, and version them.

## How to Use This Folder

- Save any prompt that produces consistently good results
- Name files clearly: `code-review.md`, `test-generation.md`, `data-extraction.md`
- When you update a prompt, note the date and what changed at the top of the file
- Include example inputs and expected outputs when possible

## Starter Prompts

These are common prompt patterns. Copy and customize as needed:

- `code-review.md` — for reviewing code quality and catching issues
- `test-writing.md` — for generating tests for existing code
- `feature-scaffold.md` — for scaffolding a new feature end-to-end
